# Otogamer Toolbox

A modern, cross-platform mobile application for synchronizing arcade game scores (Maimai DX & Chunithm) to online leaderboards.

## Features

- **Dual Platform Support**: Synchronize scores for both **Maimai DX** and **Chunithm**.
- **Dual Server Support**: Upload to **Water Fish (Diving-Fish)** and **Luoxue (LXNS)**.
- **Easy Setup**: No proxy configuration, no certificate installation. Just log in via WeChat.
- **Privacy First**: Local cookie extraction, direct upload.

## Tech Stack

- **Framework**: Flutter (Dart)
- **Networking**: Dio
- **Storage**: flutter_secure_storage
- **WebView**: flutter_inappwebview

## Getting Started

1. Clone the repository: `git clone https://github.com/NoharaYH/Otogamer-Toolbox.git`
2. Install dependencies: `flutter pub get`
3. Run the app: `flutter run`
